# Hero Alignment Summary

## What Was Standardized

- Shared background depth across homepage and subpage heroes using layered radial and linear gradients.
- Consistent vertical rhythm for hero spacing on subpages.
- Aligned heading presence through shared text shadow, line-height, and letter-spacing treatment.
- Standardized supporting text width, contrast, and line-height for improved readability.
- Added consistent subtle glow, texture, and abstract shape treatment across hero backgrounds.
- Applied a common reduced-motion-safe ambient animation system for hero surfaces.
- Kept the homepage CTA buttons visually aligned with the updated hero system.

## Shared Hero Classes

- `hero-home`
  - Used on the homepage hero.
  - Keeps the homepage layout and CTA structure while using the shared hero visual system.

- `page-hero`
  - Used on subpage headers:
    - `about.html`
    - `faculty.html`
    - `research.html`
    - `publications.html`
    - `contact.html`
    - `opportunities.html`

- Shared stylesheet:
  - `docs/styles/hero-system.css`

## Remaining Page-Specific Variations

- The homepage still includes the large CTA button pair and the larger right-side abstract orb.
- Subpages keep their simpler centered header content and use a quieter decorative variation.
- Content copy, page titles, and section structure remain page-specific and unchanged.
